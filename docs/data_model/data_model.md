# Сутності та атрибути (Movify)

## 1. User (Користувач)

- **Призначення:** зберігати дані зареєстрованих користувачів, їх профіль та налаштування.

| Атрибут   | Тип даних (приблизно) | Опис                          |
|-----------|------------------------|------------------------------|
| id        | integer (PK)          | Унікальний ідентифікатор      |
| username  | varchar(80), unique   | Логін користувача             |
| email     | varchar(120), nullable| Email (опціонально)           |
| about     | text, nullable        | Опис про себе                 |
| genres    | varchar(255), nullable| Улюблені жанри   |
| avatar    | varchar(255), nullable| Шлях до файлу аватара         |



## 2. Movie (Фільм / Мультфільм)

- **Призначення:** описувати фільм або мультфільм, який може бути доданий в улюблені.

| Атрибут    | Тип даних (приблизно) | Опис                                  |
|------------|------------------------|---------------------------------------|
| id         | integer (PK)          | Унікальний ідентифікатор              |
| title      | varchar(120)          | Назва фільму/мультфільму              |
| type       | varchar(20)           | Тип: `film` або `cartoon`             |
| genre      | varchar(80), nullable | Жанр (хоррор, комедія, драма тощо)    |
| desc       | text, nullable        | Короткий опис                         |
| image_path | varchar(255), nullable| Шлях до постера                       |



## 3. Favorite (Улюблене)

- **Призначення:** зв’язати користувача з фільмом/мультфільмом, який він додав у «Улюблені».

| Атрибут | Тип даних (приблизно) | Опис                                                        |
|---------|------------------------|------------------------------------------------------------|
| id      | integer (PK)           | Унікальний ідентифікатор запису в улюблених                |
| user_id | integer (FK → User.id) | Посилання на користувача, який додав улюблений контент     |
| movie_id| integer (FK → Movie.id)| Посилання на фільм/мультфільм, який є в улюблених          |



## 4. Session / Auth 

| Атрибут    | Тип даних           | Опис                            |
|------------|---------------------|---------------------------------|
| session_id | string / cookie     | Ідентифікатор сесії             |
| user_id    | integer             | Поточний залогінений користувач |
